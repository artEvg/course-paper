# 📚 Курсовая работа: Интернет-магазин

## 🚀 Fullstack веб-приложение для демонстрации современных технологий разработки. Реализован интернет-магазин с каталогом товаров, новостями и информацией о компании.

## 🛠️ Технологический стек

### 🔥 Frontend: React.js + React Router + TailwindCSS
### ⚙️  Backend: Node.js + Express.js 
### 🗄️  База данных: MongoDB (Atlas)

## 📁 Структура проекта

## 🚀 Быстрый запуск

### 1. Клонирование
### git clone https://github.com/artEvg/course-paper.git
### cd course-paper

### 2. Backend
### cd server
### npm install
### Создать .env с MONGO_URI, и информацией для nodemailler
### npm run server

### 3. Frontend
### cd ../client
### npm install
### npm run dev
### 🌐 Доступ: http://localhost:3000
